Claude Code's source code just leaked. Under the hood, it's a mess of vibecoded AI slop. They've also shipped 50+ amazing features in 70 days. If your engineering team is still prioritizing clean code over working product, this should terrify you.

The highlights: variable names converted to hexadecimal because of an accidental collision with an error checker. A voice-to-text module requiring three levels of workaround. A "deprecated" config that was never actually deprecated — still in production.

The CC team is doing exactly what they're telling the rest of us to do: guide agents to implement a vision and deliver code that works, not a pristine temple to order and cleanliness. AI agents can work around hex-encoded variable names. They can navigate workarounds.

What they can't work around is a team that won't ship until every edge case is beautiful.
