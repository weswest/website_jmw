name: Choice points editorial mode
description: When presenting copy options, isolate the specific decision axes with rationale for each alternative
type: feedback
---

When presenting copy alternatives, use the "choice points" format: after showing the variants, explicitly isolate the 2-4 specific decision axes (e.g., "I" vs. gerund, "hard" vs. "complex") with a one-line rationale for each option. This lets Wes make precise editorial decisions instead of choosing between whole blocks where the differences are buried.

**Why:** Wes explicitly called this out as a format he loves. It matches his decomposition mindset — break the decision into its component parts so each can be evaluated independently.

**How to apply:** Any time you're presenting 3+ copy variants, follow the variants with a "Choice points" section that names the axes of variation and the tradeoff each represents.
